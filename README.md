# belajar github lagi
Command Prompt digunakan untuk menjalankan perintah-perintah teks dan melakukan fungsi administratif tingkat lanjut pada sistem operasi Windows, tanpa perlu menggunakan antarmuka grafis (GUI). Ini sangat berguna untuk tugas-tugas seperti konfigurasi jaringan, mengelola file, memecahkan masalah, dan melakukan otomatisasi melalui script. 
Fungsi utama Command Prompt
Menjalankan perintah teks: Anda dapat mengetik perintah langsung untuk berinteraksi dengan sistem, seperti ipconfig untuk melihat informasi jaringan atau systeminfo untuk detail sistem. 
Melakukan tugas administratif: Memungkinkan Anda melakukan tindakan yang lebih mendalam yang mungkin tidak mudah dilakukan melalui antarmuka grafis, seperti format disk atau defragmentasi. 
Mengelola jaringan: Membantu dalam pemeriksaan jaringan, seperti menggunakan ping untuk menguji konektivitas atau tracert untuk melacak jalur ke server. 
Memecahkan masalah: Command Prompt adalah alat yang ampuh untuk diagnosis dan pemecahan masalah sistem yang lebih kompleks. 
Otomatisasi: Pengguna tingkat lanjut dapat menggunakan script berbasis Command Prompt untuk mengotomatiskan tugas yang berulang. 
Perbedaan dengan GUI
Command Prompt: Antarmuka berbasis teks yang membutuhkan pengguna untuk mengetik perintah. Ini menawarkan kontrol yang lebih presisi dan seringkali lebih cepat untuk tugas-tugas tertentu, serta disukai oleh para profesional TI dan pengguna tingkat lanjut.
GUI (Antarmuka Pengguna Grafis): Antarmuka visual yang menggunakan ikon, menu, dan jendela. Ini lebih intuitif dan ramah pengguna untuk sebagian besar tugas sehari-hari.
